Prueba de Evaluación Continuada 2 (PEC2)

Esta segunda PEC es totalmente práctica y agradecida a nivel de experiencia directa con la distribución de vídeo vía red, e intenta desmitificar este proceso

Competencias

● Para adaptarse en las tecnologías
● Manupular vídeo digitales aplicando principios del lenguaje audiovisual
● Para integrar y gestionar contenidos digitales
● Para implementar, poner en marcha y mantener software y hardware multimedia
● Para distribuir contenidos multimedia de manera eficiente
● Para atender consultas sobre productos multimedia

Se ha usado Visual Studio Code como editor lenguaje HTML5 y se ha integrado con Github bajo filosofia DevOps para la construccion de esta sitio.
